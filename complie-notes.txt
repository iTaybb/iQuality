Dependencies:
* Microsoft Windows >= XP
* Python == 2.7
* PyQt4 >= 4.9.4
* pywin32 >= build 218
* py2exe >= 0.6.9
* BeautifulSoup >= 4.1.3
* lxml >= 2.3.3
* mutagen >= 1.12
* mutagen IDv2.3 tags fix:
		grab fix from http://bazaar.launchpad.net/~musicbrainz-developers/picard/trunk/annotate/head%3A/picard/formats/mutagenext/compatid3.py
		or https://groups.google.com/forum/?fromgroups#!topic/quod-libet-development/krPCx4hhM_Q
		save as compatid3.py and put in mutagen's directory
* configparser python2 port >= 3.2.0
* pytest >= 2.3.4
* python-rsa >= 3.1.1
* regobj >= 0.2.2
* Inno Setup >= 5.5.1
* 7za.exe (will be downloaded if not exists)
* ffmpeg.exe (will be downloaded if not exists)

1. Make sure all the Dependencies are met, by running:
	cd C:\Scripts\iQuality\testcode
	py.test -v --pdb -k test_PyEnviron
2. For now, path must be C:\Scripts\iQuality.
3. ffmpeg must be found in env var PATH.
4. inno setup installed in 'C:\Program Files\Inno Setup 5\iscc.exe'
5. winrar file installed in 'C:\Program Files\WinRAR\WinRar.exe'
6. Run compile.bat at script's root dir.
